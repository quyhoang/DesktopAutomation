AutoChatGPT Script User Manual

Introduction

Welcome to the AutoChatGPT script user manual! This script is designed to enhance productivity for researchers by automating interactions with ChatGPT using predefined prompts.

Features

  • Pause/Resume the script using a simple hotkey.
  • Five predefined prompts for various research tasks.
  • Automated copying and pasting of selected text into ChatGPT.

Getting Started

To use the script, simply double-click the compiled executable file. The script will run in the background, and you can start using the hotkeys immediately.

Usage

Pause/Resume the Script

Press Left Alt + F2 to toggle the script on and off. This is useful if you need to temporarily disable the script.

Send Prompts to ChatGPT

Use the following hotkeys to apply a prompt to the selected text and send it to ChatGPT:

  • Ctrl + 1: Summarize the following research data
  • Ctrl + 2: Identify key insights and trends from this dataset
  • Ctrl + 3: Generate a detailed report based on the following survey results
  • Ctrl + 4: Analyze the statistical significance of the following data
  • Ctrl + 5: Provide a comprehensive literature review based on these references

Step-by-Step Guide

Step 1

Select the text you want to use with a prompt.

Step 2

Press the appropriate hotkey (e.g., Ctrl + 1).

Step 3

The script will automatically copy the selected text, combine it with the chosen prompt, and open a new ChatGPT window in your default browser.

Step 4

The combined message will be pasted into the ChatGPT input box and submitted.

Example: Imagine you have a paragraph of research data selected. By pressing Ctrl + 1, the script will generate a complete message like this:
"Summarize the following research data: [selected text]"

This message is then automatically sent to ChatGPT for processing.

Troubleshooting

If you encounter issues, try these steps:

  • Ensure no other scripts or applications are conflicting with the hotkeys.
  • Restart your computer and rerun the script.
  • Check for updates or any missing dependencies.
